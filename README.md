# 📖 Pages & Portals

> Every Story Opens a Door.

Pages & Portals is an AI-powered interactive storytelling platform that transforms simple prompts into immersive branching adventures.

Explore magical worlds, make meaningful choices, unlock hidden lore, collect artifacts, discover new pathways, and build your own living chronicle through dynamic AI-generated narratives.

---

## ✨ Features

### 🌀 AI-Powered Story Generation
Generate unique adventures from a single prompt using Groq and Llama 3.3.

Examples:
- Ancient libraries hidden between dimensions
- Forgotten kingdoms
- Magical academies
- Lost civilizations
- Portal-based mysteries
- Fantasy expeditions

---

### ⚔️ Branching Choice System

Every decision influences your journey.

Choose paths based on:

- Courage
- Knowledge
- Creativity
- Luck

Unlock different routes, discoveries, and endings depending on your choices.

---

### 📖 Chronicle Pages

A living story journal that records:

- Major events
- Story progression
- Important decisions
- Quest milestones

Presented as an animated fantasy storybook.

### 🏛️ World Lore Encyclopedia

Automatically catalog discoveries including:

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/pages-and-portals.git
cd pages-and-portals
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env` file in the project root:

```env
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=

VITE_GROQ_API_KEY=
```

### Start Development Server

```bash
npm run dev
```

## 🎯 Problem Statement

Traditional storytelling experiences are static and offer limited personalization.

Pages & Portals transforms storytelling into an interactive adventure where every decision matters, creating unique journeys shaped entirely by the player.
